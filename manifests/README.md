# Manifest Layout

This directory groups the Kubernetes and lab manifests used by the repository.

## `lab/`

Root-level manifests for the older SR-IOV, Multus, HugePages, VPP, branch pod, and client pod lab flows.

The newer Azure MANA-specific manifests live separately under [aks-dpdk-poc/manifests](../aks-dpdk-poc/manifests/README.md).
