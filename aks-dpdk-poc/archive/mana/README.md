# Archived MANA Scripts

This folder contains historical MANA-related scripts from earlier debugging phases.

They were kept because they contain useful ideas, command history, and intermediate experiments, but they are **not** the clean recommended starting point for a new engineer.

Use the organized entry points under:

- `../../scripts/mana/build/`
- `../../scripts/mana/workflows/`
- `../../scripts/mana/run/`
- `../../scripts/mana/debug/`
- `../../scripts/mana/test/`

Treat the files in this folder as reference material only.
